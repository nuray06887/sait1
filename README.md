# 🧠 Практикалық жұмыс №1 | ЖИ веб-әзірлеуде

## 📖 Жоба туралы
Бұл веб-бет әзірлеушілерге арналған ЖИ-құралдары туралы ақпараттық бет. 
Бетте келесі функционалдар бар:
- GitHub Copilot, ChatGPT, Claude AI құралдары туралы карточкалар
- DeepSeek-V3.1 (671B Cloud) моделімен чат (Ollama арқылы)
- Қараңғы/ашық тақырыпты ауыстыру
- Сұраныс санауышы
- Скролл кезінде анимация

## 🚀 Қалай іске қосу керек

### 1. Ollama орнату және іске қосу
```bash
# Ollama ресми сайтынан жүктеп алыңыз: https://ollama.com/

# Терминалда Ollama серверін іске қосыңыз
ollama serve

# Жаңа терминал терезесінде DeepSeek моделін іске қосыңыз
ollama run deepseek-v3.1:671b-cloud
